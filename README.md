# movie-rating

This is a beginner-friendly full-stack movie ranking prototype.

## Local setup

### Server

1. Open `server/`
2. Run `npm install`
3. Run `npm run dev`

The backend will start on `http://localhost:4000`.

### Client

1. Open `client/`
2. Run `npm install`
3. Run `npm run dev`

The frontend will start on a Vite development server, typically `http://localhost:5173`.

## What is included

- A minimal Express backend with a health check route.
- A minimal React + Vite frontend shell.
